# UPush-Chat
UDP chat service programmed in C

## Usage
### Compile
```bash
make all
```

### Server
```bash
./upush_server <port> <loss probability>
```

### Client 1
```bash
./upush_client <nick> <IP-adress> <port> <timeout> <loss probability>
```

### Client 2
```bash
./upush_client <nick> <IP-adress> <port> <timeout> <loss probability>
```

### Chatting
```bash
@<nick> <message>
```
